Wii

I strongly recommend GameGuy444's ripping tools, there is full documentation on this forum post.

If you have a WBFS file use the tool WBFSToISO to convert it to a Wii Disc (not ISO 9960 or compact ISO!)

100% update all the tools in here!!! especially VGMStream as inaccurate rips are possible

The latest version of VGMStream can be downloaded from here and wwPacker 1.84 (final version of wwPacker) can be downloaded here. All other programs are up to date to my knowledge.

One last thing, WiiScrubber only allows you to extract one BRSTM or whatever file at once, to combat this extract the entire DATA partition and separate the BRSTM/BRSAR/whatever files of music.

Nintendo DS

You can just put the ROM in VGMTrans and it should work fine. Just make sure to check accuracy, then right click on the music file you want to export under Detected Music Files (you should see Export To WAVE or something).

To download VGMTrans, scroll down to the table in README.md and click the link under Build Available (sorry FreeBSD!) then click the latest commit and scroll down to Artifacts then click on the build w/ your OS name. If it isn't there check the commits until you find it.

Note that these builds might be unstable and music might sound slightly off! Just be aware of this plz ty!!!!

Game Boy/Game Boy Color

Obtain a .GBS file of the game music you want to rip (or do it yourself but most games already have .GBS files ripped of them). You can get GBS files from Zophar's Domain or from joshw's website.

Once your have your .GBS, find a program that plays them (I recommend Foobar2000 with plugins). There should be a way to extract them from there. Note that you have to add fade-outs to looping music manually most of the time.

Super NES/Super Famicom

Obtain a .SPC file, or rip these by yourself. You can get SPC files from Zophar's Domain or from joshw's website.

Once you have your .SPC, get DGRFactory's SPCPlay utility, and open the file from there. For the highest quality, set Bit to 32-bit (float) and interpolation to Sinc Function. To export as .WAV, click "SAVE" and select .wav from File Type in Explorer.
